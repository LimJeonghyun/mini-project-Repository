### mini project 주제

#### 출석부 관리 시스템



### mini project 가 가지고 있는 대략적인 기능 설명

+ 학생들의 정보를 입력받아 출석 확인을 할 수 있는 프로그램을 개발한다. 

+ 간단한 예시

  #### 학번        이름   출석현황  
 ##### 22000617  임정현   3회  
 ##### 22000353  서하령   3회

+ 함수 간단히 설명   
  ##### Create : 학생 정보 생성. 

  ##### Read : 학생 정보 조회  

  ##### Update : 학생 정보 수정 & 출석 정보 업데이트  

  ##### Delete : 학생 정보 삭제  

  ##### 검색 함수 : 학생 명으로 검색, 학번 순으로 정렬, 출석 횟수로 검색  

  ##### 파일 함수 : 파일 저장 함수, 파일 읽어오는 함수  

  ##### 추가 함수 : selectMenu(), selectDataNo(), readList()
